The site's audio player looks for /audio/ambient.mp3.
If the file is missing the floating music button stays disabled.

Tips:
  - 2–4 MB is a good size target. Anything bigger slows first load.
  - The file should loop cleanly (no hard cuts). A long pad/drone works well.
  - The player starts muted and fades in to ~22% volume on first click.
